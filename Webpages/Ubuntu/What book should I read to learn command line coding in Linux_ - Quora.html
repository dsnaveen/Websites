As others have posted, there are many resources online that are free that can help you learn command line coding and scripting. Many public libraries and bookstores have books like the Linux System Administrator's Guide and  Ubuntu Unleashed that are helpful for those who like dead tree editions. Pearson Education also puts out some superb titles in this area that will get you up to speed that are highly readable textbooks.

You don't exactly need big books to learn command line coding in Linux.The two following multi-page tutorials are quite sufficient to learn complete command line coding on Linux.

Basics
   The most valuable introductory tutorial online on Linux terminal is that of Surrey.
     http://www.ee.surrey.ac.uk/Teach...

    The tutorial covers all the fundamentals to get started with the linux terminal. You will find this tutorial not only as a learning resource but also as a good reference at the beginning.

Advanced
    As you get your hands more dirty with Linux you will have more and more specific doubts on different components,concepts,new commands in Linux, which you will probably find on internet, but it doesn't result in productive learning.When I reached this point in my learning curve I found this more advanced Linux tutorial.
    http://www.tutorialspoint.com/unix/

    Though this tutorial starts with the same basics,it gives more insight into Linux.And you will learn some more advanced commands in this.This explains little more machinery behind Linux, and you will not be able to appreciate it until you have a basic understanding of Linux. So, once you reach a point, where Surrey tutorial doesn't seem to be helpful, you should start reading this  tutorial.After this, you will have your own senses to find your way.
